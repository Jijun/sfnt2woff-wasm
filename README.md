# SFNT2WOFF-wasm
run sfnt2woff on browser with webassembly


# How to build

firstly install emscripten : https://emscripten.org/docs/getting_started/downloads.html


execute :
```
git clone --recursive  https://github.com/Jijun/sfnt2woff-wasm.git
cd sfnt2woff-wasm
sh ./emscripten.sh
```
